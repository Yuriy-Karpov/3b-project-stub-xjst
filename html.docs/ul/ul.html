Пример:

Можно передать массив, где каждый элемент массива будет преобразован в li

{
   block: 'ul',
   content: [
       'item',
       'item',
       'item',
       'item'
   ]
}

а можно и элементы как обычно 

{
   block: 'ul',
   content: [
        {
            elem: 'li',
            content: 'item'
        },
        {
            elem: 'li',
            content: 'item'
        }
   ]
}
